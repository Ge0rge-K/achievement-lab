# Contributing

Thanks for helping improve this repo. Keep contributions small and meaningful.

## How to contribute
1. Open an issue using a template.
2. Create a branch: `feature/short-title` or `fix/short-title`.
3. Commit your changes with a clear message.
4. Open a pull request and link the issue.

## Co-authoring (for pair contributions)
If you pair with someone, add this line to the commit message:

Co-authored-by: Name <email>

## Commit message examples
- `docs: add glossary section`
- `feat: add checklist script`
- `fix: correct typo in README`
- `chore: update templates`
## Style
- Prefer small, focused PRs.
- Update docs when behavior changes.
